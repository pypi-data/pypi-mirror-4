pyLICORS: readme

pylicors is a Python module for predictive state estimation from continuous valued spatio-temporal data.

.. warning:: This package is in a Planning status. That means it has only very limited functionality, and function names, arguments, output, etc. WILL change in the future. Do not use this yet for any more involved analysis or automated analysis of any kind. The code you write now will break with the next releases.

pylicors has two cousins in R<www.r-project.org>: - LICORS<cran.r-project.org/web/packages/LICORS> - LSC<cran.r-project.org/web/packages/LSC>_
