
pywws.ToTwitter
***************

Post a message to Twitter

   usage: python RunModule.py ToTwitter [options] data_dir file
   options are:
    -h | --help  display this help
   data_dir is the root directory of the weather data
   file is the text file to be uploaded

This module posts a brief message to Twitter. Before posting to
Twitter you need to set up an account and then authorise pywws by
running the "TwitterAuth" program. See *Comment configurer pywws pour
poster des messages sur Twitter* for detailed instructions.
