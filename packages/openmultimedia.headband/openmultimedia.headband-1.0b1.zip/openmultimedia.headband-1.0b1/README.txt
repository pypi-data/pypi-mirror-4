Life, the Universe, and Everything
----------------------------------

This package defines an editable viewlet to add an image headband for the
site.

Don't Panic
-----------

Configure
^^^^^^^^^

- Go to 'Site Setup'.
- Select 'openmultimedia.headband settings'.
- Upload an image.

You will see the image on top of the site.
